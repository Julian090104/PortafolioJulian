Proyectos

Encriptador de texto

Proyecto hecho con HTML5, CSS3 y Javascript. Encripta cualquier texto reemplazando las vocales por alguna palabra corta. No se permite ingresar mayúsculas, acentos o cualquier caracter especial. También es posible desencriptar textos que ya han sido encriptados. Fue el primer challenge del programa Oracle Next Education con Alura Latam.

Web Barberia

Proyecto hecho con HTML5, CSS3 y JavaScript. Consiste en una página web básica para una barbería que incluye una navegación con secciones de Home, Productos y Contacto. El proyecto aborda conceptos de estructura de documentos con HTML, estilos y diseño responsive con CSS. Fue un proyecto hecho durante el programa Oracle Next Education con Alura Latam.
